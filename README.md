## 📂 Structure du Projet

Le portfolio est divisé en plusieurs modules pour une expérience utilisateur fluide :

* `index.html` : Page d'accueil avec présentation, services et vue d'ensemble du portfolio.
* `docs.html` : Documentation technique approfondie (Analyses, architectures MVC, intégrations hardware).
* `contact.html` : Formulaire de contact direct (via Formspree) et liens sociaux.
* `style.css` : Design moderne "Dark & Gold" entièrement responsive.

## 🌟 Projets Phares

### 🍔 Tommy Burger — POS & Mobile
Un écosystème complet pour la restauration rapide comprenant :
- Un logiciel de caisse Desktop tactile sous **JavaFX**.
- Pilotage d'imprimantes thermiques et tiroir-caisse via commandes ESC/POS.
- Une application mobile de supervision pour le gérant développée avec **Flutter**.

### 🤝 BarakaFood — Plateforme Solidaire
Application web Full-Stack (**PHP/MySQL**) visant à réduire le gaspillage alimentaire en mettant en relation restaurateurs et associations. Projet géré en mode collaboratif sur GitHub.

### 🏥 Gestion de Cabinet Médical
Application Desktop de gestion de patientèle avec synchronisation en temps réel sur réseau local (LAN) via **Java Swing** et **MS Access**.
